# Бот-библиотекарь для семантического поиска по текстам

Telegram-бот для индексации, семантического поиска и интеллектуальных ответов на вопросы по содержанию загруженных книг.

## Описание решения

Система реализует полнотекстовый семантический поиск с использованием векторных представлений текста и генерации ответов через LLM. Основные компоненты:

### Архитектура

**1. Индексация текстов**

- Загруженная книга разбивается на абзацы (разделение по двойным переносам строк)
- Каждый абзац сжимается до первых 400 символов для оптимизации
- Генерируется 1024-мерный эмбеддинг через multilingual-e5-large
- Эмбеддинг сохраняется в векторную БД Qdrant с метаданными (название книги, позиция в тексте)
- Метаданные сохраняются в SQLite (book_id, char_start, char_end, qdrant_id)

**2. Двухэтапный поиск**

- **Этап 1**: Векторный поиск по Qdrant
  - Запрос пользователя преобразуется в эмбеддинг
  - Поиск топ-5 наиболее близких абзацев по косинусному сходству
  - Порог релевантности: similarity > 0.5
  
- **Этап 2**: Скользящее окно
  - Внутри каждого найденного абзаца текст разбивается на чанки (по 2 предложения)
  - Для каждого чанка вычисляется точное сходство с запросом
  - Финальная сортировка по релевантности, выбор топ-5 или топ-10

**3. Генерация ответов**

- Релевантные фрагменты передаются в Mistral AI как контекст
- Промпт инструктирует модель отвечать строго на основе предоставленных цитат
- Поддерживается история диалога (контекст пользователя)

### Технологический стек

| Компонент | Технология | Назначение |
|-----------|------------|------------|
| Telegram API | aiogram 3.7.0 | Интерфейс взаимодействия с пользователем |
| Векторная БД | Qdrant (локальная) | Хранение и поиск эмбеддингов |
| Эмбеддинги | intfloat/multilingual-e5-large | Генерация 1024-мерных векторов |
| LLM | Mistral AI API | Генерация текстовых ответов |
| Реляционная БД | SQLite (aiosqlite) | Метаданные книг и абзацев |
| NLP | NLTK | Токенизация, стемминг |
| Фреймворк | sentence-transformers | Обёртка над Hugging Face моделями |

## Установка и запуск

### Системные требования

- Python 3.10 или выше
- 4 ГБ RAM (для модели эмбеддингов)
- 2 ГБ свободного места на диске

### Установка зависимостей

```bash
git clone <repository-url>
cd BOT_FOR_SADOVNIKOV

python -m venv .venv
source .venv/bin/activate  # Linux/macOS
.venv\Scripts\activate     # Windows

pip install -r requirements.txt
