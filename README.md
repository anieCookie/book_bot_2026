# Бот-библиотекарь для семантического поиска по текстам

Telegram-бот для работы с текстами книг: загрузка, индексация, поиск цитат и ответы на вопросы с использованием LLM.

## Описание решения

### Функциональность

**1. Загрузка книг**

Пользователь отправляет боту текстовый файл (.txt) и указывает название книги. Система автоматически разбивает текст на абзацы, создаёт векторные представления и сохраняет их в базу данных для последующего поиска.

**2. Поиск цитат**

Пользователь вводит поисковый запрос. Система находит наиболее релевантные фрагменты текста из всех загруженных книг и возвращает топ-5 цитат с указанием источника.

**3. Ответы на вопросы**

Пользователь задаёт вопрос по содержанию книг. Система находит релевантные фрагменты, передаёт их в LLM и генерирует развёрнутый ответ на основе текста книг.

**4. Просмотр библиотеки**

Пользователь может посмотреть список всех загруженных книг.

### Как это работает

**Индексация текстов**

1. Загруженная книга разбивается на абзацы (по двойным переносам строк)
2. Каждый абзац преобразуется в 1024-мерный вектор через модель multilingual-e5-large
3. Вектор сохраняется в Qdrant с метаданными (название книги, позиция в тексте)
4. Метаданные абзаца сохраняются в SQLite

**Поиск**

1. Запрос пользователя очищается от стоп-слов и преобразуется в вектор
2. В Qdrant ищутся топ-5 наиболее близких абзацев по косинусному сходству
3. Внутри каждого абзаца применяется скользящее окно (разбиение на чанки по 2 предложения)
4. Для каждого чанка вычисляется точное сходство с запросом
5. Результаты сортируются и возвращаются пользователю

**Генерация ответов**

1. Находятся топ-10 релевантных фрагментов
2. Фрагменты передаются в Mistral AI как контекст
3. LLM генерирует ответ строго на основе предоставленных цитат

### Технологический стек

| Компонент | Технология | Назначение |
|-----------|------------|------------|
| Telegram API | aiogram 3.7.0 | Интерфейс бота |
| Векторная БД | Qdrant | Хранение и поиск эмбеддингов |
| Эмбеддинги | intfloat/multilingual-e5-large | Векторизация текста |
| LLM | Mistral AI | Генерация ответов |
| Реляционная БД | SQLite | Метаданные книг |
| NLP | NLTK | Токенизация, стемминг |

## Установка

### Системные требования

- Python 3.10+
- 4 ГБ RAM
- 2 ГБ свободного места на диске

### Структура проекта
```bush
project/
├── main.py                 # Точка входа
├── config.py               # Конфигурация
├── requirements.txt        # Зависимости
├── user_handlers.py        # Обработчики команд
├── search_engine.py        # Логика поиска
├── ai_generators.py        # Работа с Mistral AI
├── context.py              # Контекст диалога
├── keyboards.py            # Клавиатуры Telegram
├── db/
│   └── book_repository.py  # Работа с SQLite
├── search/
│   ├── embeddings.py       # Эмбеддинги
│   └── qdrant_service.py   # Работа с Qdrant
├── text/
│   ├── text_utils.py       # Обработка текста
│   └── file_utils.py       # Работа с файлами
├── books/                  # Загруженные книги
└── qdrant_storage/         # Данные Qdrant
```

### Шаг 1. Скачивание проекта

Скачайте архив с проектом и распакуйте его в удобную директорию.

### Шаг 2. Создание виртуального окружения

Откройте терминал в папке проекта.

Linux/macOS:

```bash
python3 -m venv venv
source venv/bin/activate
```

Windows:

```bash
python -m venv venv
venv\Scripts\activate
```

### Шаг 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### Шаг 4. Загрузка данных NLTK
```bash
python -c "import nltk; nltk.download('punkt'); nltk.download('punkt_tab')"
```
